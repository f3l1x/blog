## Development

- clone this repo (`git@github.com:f3l1x/f3l1x.git`)
- install hugo (https://gohugo.io/)
- install deps `make install`

**Www**

- start devserver `make www-dev`

## Deploy

- `make build`
- `make deploy`
- yes that's all :tada:
